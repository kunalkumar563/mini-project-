# Internet Speed Test Website

A simple and accurate internet speed test website that measures:
- Download Speed (Mbps)
- Upload Speed (Mbps) 
- Ping (ms)

## Features
- Real-time speed testing
- Beautiful responsive design
- Multiple server testing for accuracy
- Verification guide with popular speed test sites

## How to Use
1. Open `index.html` in your browser
2. Click "Start Test" button
3. Wait for results
4. Use verification page to compare with other speed test sites
